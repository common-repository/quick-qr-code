Quick QR Code generates a QR code based on the current page URL
== Description ==


[Quick QR Code](https://quickqr.sharabindu.com/) generates a QR code based on the current page URL. Each QR code is unique and will be generated on Current Page Link.

== Installation ==

Go to the Plugins Menu in WordPress and select Plugins > Add new
Search for “ Quick QR Code”
Click “Install” and “activate”.
